Política de privacidad

Esta política explica qué datos procesa el bot (“Servicio”) y con qué propósito.

Datos que podemos procesar

  • Identificadores técnicos: IDs de servidor, canal, mensaje y usuario para operar comandos y automatizaciones.
  • Preferencias del servidor: configuraciones (p. ej., roles/IDs) almacenadas en archivos JSON o base de datos.
  • No almacenamos el contenido completo de mensajes salvo que lo configures explícitamente para funciones concretas (p. ej., tickets o logs).

Base legal y finalidad

  • Proveer funcionalidades del bot (moderación, música, roles, tickets, etc.).
  • Mejorar la estabilidad y seguridad mediante registros de errores.

Retención

Conservamos datos mínimos el tiempo necesario para operar el Servicio y cumplir con requisitos legales o de moderación del servidor.

Compartición

No vendemos datos. Podemos usar proveedores de infraestructura (p. ej., hosting, GitHub Pages, proveedores de audio) para prestar el Servicio.

Derechos

Puedes solicitar eliminación de configuraciones asociadas a tu servidor. Contacto al email del administrador del bot.
